# Shadow Mirror (MVP)

Small-scale project to mirror requests from a production endpoint to a shadow/UAT endpoint and compare responses.

## Tech stack
- Backend: Spring Boot (Java 17) with WebClient
- DB: PostgreSQL
- Frontend: React (Vite)
- Build: Maven (backend), npm (frontend)
